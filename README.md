## 🛠️ Trik GitHub
Tips dan trik menarik untuk mempercantik README dan proyek GitHub menggunakan GIF dan animasi.

---

## 📜 Cara Menggunakan
1. Pilih GIF yang diinginkan dari daftar di atas.
2. Klik kanan pada gambar dan pilih "Salin URL Gambar".
3. Tempel URL dalam format berikut di README.md Anda:
   ```markdown
   ![Deskripsi Gambar](URL_GAMBAR)
   ```
4. Commit dan push perubahan ke repository GitHub Anda!
